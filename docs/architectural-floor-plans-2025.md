在本專案的定位摘要：

- **B1**：停車空間與後勤區，不直接參與隔代共學流程。
- **1F 長者日照中心**：本系統主要的長照節點，結合失智照護專區、一般日照活動區、共用餐廳與無障礙庭園，是多數隔代共學情境發生的「起點與舞台」。
- **2F 公共托嬰中心**：承接 0–2 歲嬰幼兒照護，與 1F 日照中心透過共煮、共食、故事時間等活動形成「垂直互動」。
- **3F 家庭支持服務中心**：作為家長諮商、親職教育與社區營造樞紐，同時是志工訓練與方案設計的基地。
- **4F 青少年社區活動空間**：國中生以上青少年志工、課輔與專題空間，讓年輕世代成為隔代共學中的「橋樑」。

前端頁面中的「館內平面圖 + 即時空間利用率」卡片，即是依照上述樓層分工與 GitHub 設計稿，做出的簡化視覺化呈現。
